# Mini-c-logical-projects
A collection of small logic-based programs that demonstrate fundamental programming concepts such as conditional statements, mathematical calculations, and user input handling.
# 🧠 Collection of Logic-Based Programs

This repository contains a collection of **independent C++ programs** created to practice basic programming logic and problem-solving.

> ⚠️ These programs are **not related to each other**. Each one is a separate task demonstrating different concepts.

---

## 🚀 Programs Included

### 🔞 Age Verification

Checks whether a user meets a required age condition.

---

### 🧮 Calculator

Performs basic arithmetic operations:

* Addition
* Subtraction
* Multiplication
* Division

---

### 🧮 Calculator (Functions Version)

A calculator implemented using functions to improve structure and code organization.

---

### 🔢 Greatest Common Divisor (GCD)

Calculates the greatest common divisor of two numbers using mathematical logic.

---

### 📐 Square Root Calculator

Computes the square root of a number.

---

## 🧠 Concepts Demonstrated

* Conditional statements (if/else)
* Functions
* User input handling
* Mathematical operations
* Basic algorithms

## ▶️ How to Run

All programs are written in C++, so they must be compiled before running.

```bash id="y91jmk"
# Age Verification
g++ age_varification.cpp -o age
./age

# Calculator
g++ calculator.cpp -o calc
./calc

# Calculator (Functions Version)
g++ calclator_with_function.cpp -o calc_func
./calc_func

# Square Root
g++ Square_Root.cpp -o sqrt
./sqrt

# GCD
g++ Greatest_Common_Divider.cpp -o gcd
./gcd
```
